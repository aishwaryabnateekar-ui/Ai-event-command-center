# 🚀 EventIQ AI – Smart Event Command Center

## 📌 Overview
EventIQ AI is an AI-powered dashboard designed to enhance the experience of physical events by providing real-time insights, crowd analysis, and smart recommendations.

---

## 🎯 Problem
Managing large physical events is challenging:
- Difficult to track attendance in real time  
- Overcrowding issues  
- Lack of actionable insights for organizers  

---

## 💡 Solution
EventIQ AI provides a smart dashboard that:
- Monitors event participation  
- Analyzes crowd status  
- Suggests AI-based actions to improve event flow  

---

## ✨ Features
- 📊 Real-time Event Dashboard  
- 👥 Attendance Tracking (Guests, Attendees, Absent)  
- 🚦 Status Indicator (Smooth / Moderate / Overcrowded)  
- 🤖 AI Insights & Recommendations  
- 📈 Visual representation of attendance  

---

## 🧠 AI Insights Examples
- "Crowd spike expected at 3 PM"  
- "Attendance at 72% - good engagement"  
- "Open additional hall to reduce congestion"
